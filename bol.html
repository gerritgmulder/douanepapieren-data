Betalingen doorvoeren in Logic4

0 van 0 credit-regels zijn succesvol gematcht met een Logic4-order. Daarvan kunnen de betalingen direct gekoppeld worden via /v3/Orders/AddPayment — geen handmatige actie meer per regel.

⚠ De 2 debet-regels (verkoopprovisie + Bol-overschrijving) blijven Excel-export — die kunnen niet via API (boekhoud-module zit niet in scope). Osman voert die twee handmatig in Logic4 in.
